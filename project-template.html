Project 1: Clínica Alivio Brand Identity

Role: Designer

Tools: Adobe InDesign, Illustrator, Photoshop

Timeline: Spring 2026 semester project

This project focused on developing a complete visual identity for Clínica Alivio, a healthcare-focused brand. The goal was to create a design system that communicates trust, clarity, and professionalism while still feeling approachable and modern.

The identity system includes logo design, typography selection, color palette development, and applied brand materials that demonstrate consistency across multiple touchpoints.

Project 1: Brand Identity

The Challenge:

One of the main challenges was creating a design that felt trustworthy without appearing overly clinical or generic. Many healthcare brands rely heavily on predictable visual styles, so the goal was to create something distinctive while still appropriate for the audience.

Typography and color choices were adjusted throughout the process to improve clarity and ensure readability across different sizes and formats.

The Solution:

The final design uses a balanced layout structure, controlled color palette, and consistent typography hierarchy to create a recognizable visual system. Spacing and alignment were refined to improve readability and maintain a professional appearance.

The result is a brand identity that feels modern, organized, and adaptable across multiple applications.

Project 2: OOH Advertising Campaign

Role: Designer

Tools: Adobe InDesign, Illustrator, Photoshop

Timeline: Spring 2026 semester project

This project focused on developing an out-of-home advertising campaign designed for high visibility placements such as billboards, transit shelters, and poster boards.

The objective was to create bold, simplified visuals that communicate quickly and effectively to audiences moving through physical environments.

Project 2: OOH Advertising Campaign

The Challenge:

Designing for large-scale formats requires careful consideration of spacing and readability. Elements that appear balanced on a computer screen may not function the same way when scaled to billboard size.

Typography size, spacing, and contrast were refined to improve visibility and maintain clarity across different formats.

The Solution:

The final campaign uses simplified messaging, strong contrast, and clear focal points to communicate effectively in a short amount of time. Each format maintains consistency while adapting to different dimensions and viewing distances.

The result is a cohesive campaign system that demonstrates an understanding of large-scale design principles and audience engagement strategies.
