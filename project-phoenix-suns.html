Sports Marketing Campaign

Phoenix Suns

The Challenge

Create a high-impact promotional graphic for a basketball game that captures attention immediately and communicates essential game information — energy, movement, and event details — through a clear visual hierarchy.

The Solution

A bold Game Day poster using strong typography, dramatic lighting, and warm color contrast. The design system was adapted across vertical story, square post, OOH display, and mobile formats to demonstrate real-world campaign consistency.
